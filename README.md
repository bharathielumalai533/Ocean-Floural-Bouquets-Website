# Ocean Floral Bouquets Website

A modern and responsive floral bouquet showcase website designed to present elegant flower collections with a visually appealing user interface and seamless browsing experience.

## 🌐 Live Demo

https://ocean-floural-bouquets-website.vercel.app/

## 📖 Overview

Ocean Floral Bouquets is a React-based web application developed to showcase a variety of floral arrangements and bouquet collections. The website focuses on providing an attractive design, smooth navigation, and a responsive user experience across all devices.

## ✨ Features

* Responsive and mobile-friendly design
* Beautiful floral bouquet showcase
* Modern and elegant user interface
* Product display sections
* Customer reviews section
* Smooth navigation experience
* Optimized performance

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* Tailwind CSS

## 🚀 Installation

```bash
git clone https://github.com/bharathielumalai533/Ocean-Floural-Bouquets-Website.git
cd Ocean-Floural-Bouquets-Website
npm install
npm start
```

## 🎯 Purpose

This project was developed to create an engaging online platform for displaying floral bouquet collections and enhancing the digital presence of a flower business through an attractive and responsive web experience.
